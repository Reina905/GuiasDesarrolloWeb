Desarrollo Web I

Descripción

Este curso enseña los fundamentos de HTML, CSS y JavaScript para aplicaciones web atractivas, funcionales e interactivos, comprendiendo su arquitectura front-end, las mejores prácticas de diseño y algunas prácticas desventajosas en el desarrollo de software.

  • UNIDAD I: Introducción a HTML
  • UNIDAD II: Introducción a CSS
  • UNIDAD III: Maquetación de sitios web
  • UNIDAD IV: Frameworks de diseño web responsivo (Bootstrap)
  • UNIDAD V: Introducción a JavaScript
  • Unidad VI: El modelo de objetos del documento (DOM) y JavaScript asíncrono

Guia 1: Introducción y conceptos básicos de HTML y CSS

Descripcion: La guía aborda la estructura básica de un documento web con HTML, la aplicación de estilos CSS embebidos y externos, y el uso de selectores y propiedades CSS para dar formato a textos y fuentes.

Ejemplos:

  1. Estilos embebidos: Usando estilos CSS embebidos en las etiquetas HTML
  2. Ejemplo 2: Incluyendo reglas css mediante la etiqueta style
  3. Paises: Usando reglas CSS definidas en archivos externos

Ejercicios:

  1. Ejercicio 1: cambiar los estilos embebidos a un archivo externo
  2. Mi programa favorito cuando era pequeña

Guia 2: Selectores CSS avanzados y modelo de caja

Descripcion: La guía aborda selectores CSS avanzados, pseudo-clases y pseudo-elementos, y el uso del Modelo de Cajas para estructurar páginas web.

Ejemplos:

  1. Usando selectores CSS avanzados
  2. Diseñando un tablero para damas con la pseudo-clase :nth-child()

Ejericio:

  1. Ejercicio de autobiografia

Guia 3: Posicionamiento, visualización y maquetación de sitios

Descripcion: La guía aborda posicionamiento y visualización de elementos en CSS, así como la maquetación de sitios con diseños fijos y fluidos.

Ejemplos

  1. Usando posicionamiento y visualización
  2. Diseñando un menú horizontal multinivel
  3. Construyendo un diseño fijo

Ejercicios

  1. Resultado 1: Usando posicionamiento y z-index
  2. Resultado 2: Usando posicionamiento y z-index
  3. Ejemplo 1 modificado, con el footer y header estaticos
  4. Construyendo un diseño liquido a partir del ejemplo 3

Guia 4: Introducción al Responsive Design

Descripcion: Esta guía permite diseñar páginas web responsivas, aplicando correctamente la meta-etiqueta viewport y definiendo media queries para el diseño responsivo.

Ejemplos

  1. Ejemplo de Media Queries
  2. Agencia de Viajes: Diseño Responsivo

Fortalezas

Al tomar este curso, mejoré muchas habilidades, como la creación del front-end de aplicaciones web. Además, aprendí a mantener siempre el orden, la funcionalidad y la interactividad en ellas mediante HTML y CSS, y considero que también he mejorado un poco mi lógica de programación. Por otro lado, nunca había programado en JavaScript, por lo que esta es una nueva habilidad que estoy adquiriendo.

Oportunidades

Considero que una de las cosas más importantes que estoy aprendiendo, y que seguiré aprendiendo a profundidad, es saber usar bien JavaScript, ya que no solo se utiliza para el desarrollo de páginas web, sino también para las bases de datos NoSQL (los JSON), para la comunicacion con los servidores, APIs, desarrollo móvil, entre otros.

Debilidades

Una de mis mayores debilidades es que, si bien sé CSS muy bien, a veces me tardo mucho al escribir el codigo porque siempre analizo mucho sobre qué debo hacer. Por otro lado, aún estoy aprendiendo JavaScript, así que a veces me confundo con la sintaxis.

Amenazas

Considero que un obstáculo hubiera podido ser no organizar bien mi agenda para dedicar, aunque sea un poco de tiempo, a estudiar y hacer las guías de ejercicios. Pero esto no pasó, y siempre hago espacio para aprender más sobre el desarrollo web.
